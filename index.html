Handling events

  • Interactions create events which trigger code and it responds to users
  • UI events occur when a user interacts with the browser's user interface rather than the webpage Eg. load, unload, error resize, scroll
  • Keyboard events occur when a user interacts with the keyboard Eg. keydown,keyup,keypress
  • Mouse events occur when user interacts with a mouse Eg. click,dblclick, mousedown, mouseup, mousemove , mouseover, mouseout
  • Focus events occur when an element gains or loses focus Eg. focus, blur
